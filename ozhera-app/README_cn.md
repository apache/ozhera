# ozhera-app
ozhera-app 是管理监控应用基础数据的微服务，是整个Apache OzHera(incubating)监控体系的数据服务中心，
为链路追踪、日志、指标监控提供一致性的统一应用数据，同时管理着全局的配置信息。
